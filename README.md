# adacezvem
Ada Cezvem QR Menu Project.
This project is a web-based digital menu solution designed for Ada Cezvem. It allows customers to access the menu instantly by scanning a QR code, providing a seamless and contactless experience.

🚀 Features
Instant Access: Customers can view the menu on their mobile devices without downloading any app.

Always Up-to-Date: Prices and items can be updated in real-time without changing the physical QR codes.

Responsive Design: Fully optimized for mobile, tablet, and desktop screens.

🛠️ Tech Stack
Frontend: HTML5, CSS3

Deployment: GitHub Pages

Integration: Static QR Code

📖 How It Works
The menu is hosted as an index.html file on GitHub Pages.

A static QR code points directly to the live URL of the repository.

Any updates made to the code are immediately reflected on the digital menu, ensuring the QR code remains permanent.
